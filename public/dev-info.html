How the tool works

  1. You paste Lua code locally inside your browser.
  2. The backend (Node + Express) receives your snippet over HTTPS on your own machine.
  3. The server writes the snippet to a temporary file, runs lua ./Prometheus-master/cli.lua with your selected preset, and waits for Prometheus to finish.
  4. The obfuscated output is streamed back and displayed instantly.
  5. The temporary directory is deleted right after the response is sent.

Privacy stance

This project does not log, upload, or retain your scripts. Everything happens on the same machine you run the server on. There are no analytics, trackers, remote calls, or hidden services - just Prometheus running locally.

If you need to audit the behavior, the entire stack lives in this folder. Skim server.js to see how Express spawns Prometheus and removes the temporary files.
